Overview

This project presents an interactive HR Attrition Analysis Dashboard developed using Microsoft Excel.

The dashboard provides insights into employee attrition trends, workforce demographics, department-wise performance, and key HR metrics to support data-driven workforce planning and retention strategies.

It transforms raw HR data into meaningful insights for management decision-making.

Project Purpose

The purpose of this dashboard is to help HR managers and business leaders:

Monitor employee attrition and workforce stability

Analyze attrition rate by department, job role, and education

Understand demographic patterns (age, gender, marital status)

Support strategic workforce planning and retention initiatives

Tech Stack

Microsoft Excel – Dashboard design and visualization

Pivot Tables & Pivot Charts – Data aggregation and analysis

Excel Formulas – KPI calculations (Attrition Rate, Active Employees, etc.)

Slicers – Interactive filtering

Data Cleaning & Structuring – Excel data transformation

Data Source

Sample HR employee dataset (portfolio demonstration)

Dataset includes:

Employee ID

Age

Gender

Education

Job Role

Department

Marital Status

Attrition (Yes/No)

Salary & Workforce details

Note: This dashboard uses sample data for learning and portfolio purposes.

Business Problem

Employee attrition significantly impacts organizational performance, hiring costs, and operational stability.

Without proper analytics, it becomes difficult to:

Identify departments with high attrition

Understand demographic risk patterns

Monitor workforce stability

Develop targeted retention strategies

Dashboard Goals

Provide a centralized HR performance overview

Monitor attrition rate and active employees

Identify department-wise and role-wise attrition

Analyze demographic factors influencing attrition

Enable interactive filtering for deeper analysis

Key Dashboard Features / Highlights

KPI Cards:

Total Employees

Attrition Count

Active Employees

Attrition Rate

Average Age

Gender distribution analysis

Education-wise attrition breakdown

Job role-wise attrition comparison

Department-wise attrition percentage

Age group attrition analysis

Marital status attrition analysis

Interactive filters (Education, Department)

Key Visuals & Why They Were Used
KPI Cards

Used to provide a high-level summary of workforce status and attrition metrics.

Donut Chart (Gender Distribution)

Helps visualize workforce composition by gender.

Bar Charts (Education, Marital Status)

Used to compare attrition distribution across categories.

Pie Chart (Department-wise Attrition)

Helps identify departments contributing most to employee turnover.

Column Chart (Age-wise Attrition)

Shows age group patterns and potential workforce risk segments.

Treemap (Attrition by Job Role)

Used to highlight which roles contribute most to attrition.

Key Insights

Certain departments show significantly higher attrition rates

Mid-career age groups contribute more to workforce turnover

Job roles such as sales-related positions show higher attrition

Education level and marital status impact attrition patterns

Conclusion

This HR Attrition Dashboard demonstrates how Excel can be used as a powerful analytics tool for HR decision-making.

It provides actionable insights into employee turnover patterns and supports strategic planning to improve workforce retention and organizational stability.
